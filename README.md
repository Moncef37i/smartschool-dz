# 🎓 SmartSchool DZ

**SmartSchool DZ** is a premium, modern School Management System (SaaS Dashboard) built to streamline educational administration. It provides a centralized platform for managing students, faculty, academics, and school operations with a focus on performance and user experience.


## ✨ Key Features

- **📊 Comprehensive Dashboard**: Visualize school health with real-time statistics and interactive charts.
- **👥 Student Management**: Effortlessly manage student profiles, enrollments, and personal data.
- **👨‍🏫 Teacher Portal**: Organize faculty records, assignments, and contact information.
- **🏫 Class Administration**: Structure your school with detailed class and department management.
- **📅 Attendance Tracking**: Monitor daily attendance for students and staff with ease.
- **🏆 Grade Management**: Input and track student performance, exams, and grading history.
- **💰 Fee Tracking**: Manage school finances, tuition status, and payment records.
- **🕒 Interactive Timetable**: A clean, visual representation of daily schedules and class timings.
- **📢 Announcements System**: Keep everyone informed with school-wide updates and notifications.
- **🌓 Dark/Light Mode**: Premium design system with full support for user theme preferences.
- **📱 Fully Responsive**: Optimized for desktops, tablets, and mobile devices.

## 🚀 Tech Stack

- **Core**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **Routing**: [React Router DOM](https://reactrouter.com/)
- **State Management**: React Context API + LocalStorage for persistence

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/moncef37i/smartschool-dz.git
   cd smartschool-dz
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🌐 Deployment

The project is configured for deployment on **GitHub Pages**. To deploy your version:

```bash
npm run deploy
```

Visit the live demo: [https://moncef37i.github.io/smartschool-dz/](https://moncef37i.github.io/smartschool-dz/)
